Тут знаходиться два файли для роздільної компіляції в яких значення залежать один від одного та не можуть функціонувати окремо.
Спочатку потрібно написати команду яка скомпілює їх окремо в байтний код: g++ -c main.cpp func.cpp 
Файли отримають рощширення .о , потім потрібно злінкувати цих два файли: g++ -o program main.o func.o
Звідси ми отримали готову програму program яку достатньо просто відкрити: ./program
